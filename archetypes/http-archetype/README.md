### How to use it :

```
mvn archetype:generate \
        -DarchetypeGroupId=io.datakernel                  \
        -DarchetypeArtifactId=datakernel-http-archetype   \
        -DarchetypeVersion=3.0.0-SNAPSHOT                 \
        -DgroupId=<group id>                              \
        -DartifactId=<artifact id>                        \
        -DmainClassName=<your main class name> 
```
